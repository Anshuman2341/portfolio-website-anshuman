# Portfolio Website - Anshuman Singh

## 🚀 About the Project
This is my **personal portfolio website** showcasing my skills, experience, and projects. It features a clean, modern UI with responsive design, smooth animations, and interactive elements.

## 🌟 Features
- 🖥️ **Responsive Design** - Works across all devices (desktop, tablet, and mobile)
- 🎨 **Modern UI** - Styled with CSS for a professional appearance
- 🏗️ **Sections Included**:
  - About Me
  - Work Experience
  - Education
  - Skills
  - Contact Info
- ⌨️ **Typed.js Integration** - Dynamic text animation
- 📄 **Downloadable Resume**
- 🔗 **Links to GitHub & LinkedIn Profiles**

## 🛠️ Technologies Used
- **HTML5**
- **CSS3**
- **JavaScript** (Typed.js for text animations)
- **Google Fonts**

## 📂 Folder Structure
```
portfolio-website/
```
